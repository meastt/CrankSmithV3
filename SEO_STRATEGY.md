# SEO Strategy for CrankSmith

## 1. Core Keywords
- **Primary**: Bike Builder, Bicycle Compatibility Checker, Cycling Tool, Gear Ratio Calculator, Tire Pressure Calculator.
- **Secondary**: Road Bike Builder, Gravel Bike Builder, MTB Builder, Bike Part Compatibility, Weight Weenies Calculator.
- **Long-tail**: "Will these wheels fit my frame?", "SRAM AXS compatibility chart", "Optimal tire pressure for gravel", "Gear inches calculator 700c".

## 2. Metadata Strategy
### Global Metadata (`layout.tsx`)
- **Title**: CrankSmith | The Ultimate Bicycle Builder & Compatibility Tool
- **Description**: Build your dream bike with confidence. Check part compatibility, calculate gear ratios, optimize tire pressure, and estimate weight. The precision tool for serious cyclists.
- **OpenGraph**: Use a high-quality screenshot of the builder or a generated social card.

### Page-Specific Metadata
- **/builder**: "Custom Bike Builder - Check Compatibility & Weight"
- **/performance**: "Drivetrain Lab - Gear Ratios & Speed Calculator"
- **/tire-pressure**: "Tire Pressure Calculator - Road, Gravel, MTB"
- **/weight**: "The Scale - Bike Weight Weenies Calculator"

## 3. Technical SEO
- **Sitemap**: Generate `sitemap.xml` dynamically based on routes and (eventually) public builds.
- **Robots.txt**: Allow indexing of main tools, disallow user-specific private pages if any.
- **Structured Data**: Add JSON-LD for "SoftwareApplication" or "WebApplication".

## 4. Content Strategy (Future)
- **Guides**: Create "How-to" guides for using the tools (e.g., "How to measure rim width", "Understanding gear inches").
- **Blog**: Articles on bike tech trends, compatibility deep dives.
- **Part Pages**: Dynamic pages for specific parts (e.g., `/parts/shimano-dura-ace-rd-r9250`) with compatibility info.
